 == MODEL PROMPT ===
 Build a student grade tracker in Python.

Task:
Take a student's name and marks in 5 subjects, calculate their average, assign a grade, determine pass/fail, and save the result to a .txt file.

Input rules:
- Ask: "Enter student name:" — must be non-empty alphabetic string. If not, show error and re-prompt.
- Ask: "Enter marks for 5 subjects separated by commas (e.g. 80,90,70,60,50):"
- Each mark must be an integer between 0 and 100 inclusive. If not, show error and re-prompt the entire marks line.

Logic:
- Calculate average: sum of 5 marks / 5
- Round average to nearest integer using round()
- Assign grade using these exact rules (no gaps, no overlaps):
  >= 81 → A (Pass)
  >= 66 and <= 80 → B (Pass)
  >= 51 and <= 65 → C (Pass)
  >= 35 and <= 50 → D (Pass)
  < 35 → F (Fail)
- Pass if grade is A, B, C, or D. Fail if grade is F.

Output:
- Print: "Student: John | Average: 70 | Grade: B | Result: Pass"
- Save to a .txt file named after the student (e.g. John.txt) with this exact format:
  Student Name: John
  Marks: 80, 90, 70, 60, 50
  Average: 70
  Grade: B
  Result: Pass

Process:
- Plan first, implement only when confident
- Verify with: John, marks 80,90,70,60,50 → average 70, grade B, result Pass, saved as John.txt
- Also test edge case: marks 30,30,30,30,30 → average 30, grade F, result Fail
